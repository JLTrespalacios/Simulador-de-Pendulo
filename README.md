### ACA FINAL — Física Mecánica y Laboratorio
Este proyecto es una suite de herramientas web interactivas para el aprendizaje de física mecánica, desarrollada con tecnologías web modernas (HTML5, CSS3, JavaScript). Consta de tres módulos principales:

1. Simulador de Péndulo Simple :
   
   - Simulación Física Avanzada : Utiliza el método numérico Runge-Kutta de 4to orden (RK4) para resolver las ecuaciones de movimiento con alta precisión.
   - Variables Reales : Considera factores como la gravedad, longitud de la cuerda, masa, y fuerzas disipativas (fricción del aire y del pivote).
   - Análisis : Muestra en tiempo real el periodo, frecuencia, fuerzas resultantes y la conservación (o pérdida) de energía mecánica. Permite exportar los datos a CSV.
2. Simulador de Plano Inclinado :
   
   - Dinámica de Bloques : Simula el deslizamiento de un objeto sobre una superficie inclinada.
   - Fuerzas Interactivas : Permite ajustar el ángulo de inclinación, coeficiente de fricción ($\mu$) y masa.
   - Visualización : Muestra vectores de fuerza y gráficas de energía cinética/potencial en tiempo real.
3. Calculadora Einstein Interactiva :
   
   - Herramienta Matemática : Una calculadora científica completa (trigonometría, logaritmos, raíces) basada en la librería math.js .
   - Experiencia de Usuario : Cuenta con una interfaz temática "Einstein" que incluye efectos visuales y una función de texto a voz que lee los resultados de las operaciones.
